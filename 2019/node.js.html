# Convention simplifiée de formation professionnelle (article L 6353-1 du code du travail) **Entre les soussigné·e·s** : 1. l'organisme de formation `dtc innovation` dont la déclaration d'activité est en cours d'enregistrement auprès du préfet de région Nouvelle-Aquitaine d'une part, 2. et la personne morale **{{trainee}}** d'autre part, est conclue la convention suivante, en application des dispositions du livre IX du code du travail portant sur l'organisation de la formation professionnelle continue dans le cadre de l'éducation permanente. ## Objet de la convention L'organisme `dtc innovation` organise l'action de formation suivante : **{{title}}**. L'objectif de la formation "{{title}}" est d'amener les participant·e·s à intégrer de nombreuses techniques de programmation, de manière à ce qu'ils ou elles puissent s'en servir de manière réflexe, dans leur travail quotidien, quand la situation l'exige. Ce cours est présenté en français par un intervenant `dtc innovation` sur des sessions hebdomadaires, sous forme magistrale et interactive. Les participant·e·s sont invité·e·s au fil des sessions à mémoriser, à s'approprier, puis à présenter au reste du groupe des exercices de programmation de difficulté croissante, exercices s'attachant à des problématiques diverses rencontrées dans le développement logiciel. Les participant·e·s ont la possibilité pendant et entre chaque session de discuter avec l'intervenant `dtc innovation`, expert en programmation, en techniques de développement incrémental de logiciel et en transmission de compétences par la pratique. Cette formation s'adresse à des développeur·ses web qui ont un historique de développement dans un autre langage que JavaScript (PHP, Ruby, C#, etc.) et qui ont envie de devenir des professionnel·le·s reconnu·e·s sur les technologies JavaScript et Node.js. Il est attendu comme pré-requis que les participant·e·s aient une connaissance préalable et pratique des éléments suivants : - développement d'applications web dynamiques - développement et utilisation d'API REST - développement d'interfaces web avec HTML, CSS et JavaScript - utilisation d'un éditeur de code pour modifier des programmes existants - gestionnaire distribué de versions Git Type d'action de formation (article L 6313-1 du code du travail : adaptation, promotion, prévention, acquisition, entretien ou perfectionnement des connaissances) : **acquisition et perfectionnement de connaissances**. - Dates : 25/04/2019, 2/05/2019, 9/05/2019, 16/05/2019, 22/05/2019, 29/05/2019 - Durée : 6 sessions de 2 heures (binômage) + 3 sessions de 30 minutes (rétrospective), soit **13 heures et demi** de formation - Horaires : les sessions sont organisées à 10h, sauf arrangement communiqué dans les 3 jours qui précèdent - Lieu : vidéoconférence sur Internet (Skype, Jitsi, Zoom.us) ## Effectif formé L'organisme `dtc innovation` formera **{{trainee}}**. ## Plan et support de cours Chaque participante et participant aura accès à un serveur web sur lequel seront repris les exercices et les notions abordées en cours. ### JavaScript - les primitives et types de données - programmation asynchrone (callbacks, promesses, `async`/`await`) - travailler avec des structures de données : opérateur spread, destructuration, assignation - copier des valeurs (par référence, par copie) - les objets `Map` et `Set` - les symboles - architecturer son code sans effet de bord (variable globale vs. modules) - programmation fonctionnelle ### Node.js - solutions CMS clé-en-main (à la Drupal) - exploration des modules natifs - programmation événementielle - architecturer son code de manière modulaire - écriture de tests unitaires (_Test Driven Development_) - tests automatisés de parcours utilisateurs - déploiement automatisé (intégration continue, déploiement continu) - authentification des utilisateurs (JSON Web Token) ### Développement web - outiller le partage de modules entre Node.js et une page HTML - optimiser les ressources graphiques (CSS, fontes, images bitmap, images vectorielles) - programmation avec un arbre à états (_design-pattern_ Redux) - programmation avec des composants graphiques pilotés par la donnée (React, Vue.js) - partager les composants de rendu d'affichage entre le serveur et le client ### Agilité logicielle - binômage - intégration continue - principes de classification, d'organisation et de prises de décision collectives avec une gestion fine des prises de parole - principes d'amélioration continue de manière itérative, pilotée par des contraintes de temps, de budget et de valeur produit - rédaction d'actions de développement ("_user stories_") pour expliciter les attentes - animation de réunion de planification et de priorisation des actions de développement - animation de réunion de rétrospective pour capter les sources de réussite et d'amélioration du travail en équipe - conception de plans d'itération avec des outils numériques (Trello, GitHub Projects, GitLab Projects, Taïga) ## Dispositions financières En contrepartie de cette action de formation, l'employeur s'acquittera des coûts suivants.
